# iceberg-bioimage schema

This directory includes information about the iceberg-bioimage schema.
The schema is based on the [OME data model](https://github.com/ome/ome-model/blob/master/specification/src/main/resources/released-schema/2016-06/ome.xsd) and [REMBI modules](https://doi.org/10.1038/s41592-021-01166-8).
We use [LinkML](https://linkml.io/) to specify the schema with plans for integration in related projects.
